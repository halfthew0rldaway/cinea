# Cinea

Cinea is a premium, modern web application designed to deliver a visually stunning, Netflix-inspired movie streaming and discovery experience. Built with a strict focus on high-end aesthetics and seamless UI/UX, the platform provides users with an immersive way to browse, search, and stream their favorite movies and TV shows.

## Overview

The application serves as a dynamic streaming frontend that connects to multiple data sources to provide a rich media library. It goes beyond a simple minimum viable product by incorporating polished micro-animations, a responsive design, and a unique personality-driven onboarding flow.

## Key Features

- **Premium UI/UX Design**: A highly polished, responsive interface featuring deep dark modes, harmonious color palettes, modern typography, and smooth interactions that rival top-tier streaming platforms.
- **Integrated Web Player**: Seamless video playback functionality utilizing standard embeds and robust streaming providers (like vidsrc.net), ensuring cross-browser compatibility and a frictionless viewing experience.
- **Dynamic Media Metadata**: Automatically retrieves, normalizes, and displays high-quality posters, backdrops, and movie details using integrations with TMDB and Watchmode APIs.
- **Unique Onboarding Experience**: Features a custom, personality-driven (and slightly sarcastic) welcome modal that guides new users into the platform with a touch of humor.
- **Robust API Management**: Designed with production reliability in mind, it handles secure API requests, adheres to strict rate limits via aggressive caching strategies, and proxy forwards requests reliably.
- **Modern Tech Stack**: Built with React and Vite, ensuring lightning-fast performance, optimized builds, and a highly responsive single-page application feel.
